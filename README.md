# Mytube
This project is a static clone of the YouTube homepage created using pure HTML and CSS. It demonstrates my understanding of CSS layouts, Flexbox/Grid, responsive design, and UI structuring.

YouTube Homepage Clone

A frontend project that recreates the YouTube homepage UI using HTML and CSS only.

Key features:

-> Responsive layout

-> Header, sidebar, and video grid design

-> Flexbox & CSS Grid usage

-> Clean and structured HTML

This project helped me strengthen my understanding of modern CSS techniques and UI design principles.
